Bootstrap ən məşhur css frameworklərindəndir. Onun köməkliyi ilə asan bir şəkildə web səhifələr düzəltmək olar. Bootstrap classlarla işləyir hər bir elementə css tətbiq etmək üçün həmin elementə istədiyimiz formaya uyğun classlar verməliyik həmin classların işləməsi üçündə bootstrap linklərini səhifəyə əlavə etməliyik. Bootstrap kimi paketi özümüzdə yarada bilərik eyni class məntiqi ilə cssde classlara stil tətbiq edib sonra onlari htmldə istifadə edə bilərik.
